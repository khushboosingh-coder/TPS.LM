Powerful Features
Everything you need for efficient leave management

📝

Leave Requests

Employees can easily submit leave requests with detailed information about leave type, dates, and reasons.

✅

Smart Approvals

Managers review, approve, or reject requests with detailed comments and audit trails.

📊

Balance Tracking

Real-time tracking of leave balance, used days, and pending requests per employee.

🔐

Secure Auth

JWT-based authentication with OAuth 2.0 support for secure single sign-on.

📈

Reports & Analytics

Comprehensive reports on leave usage, department statistics, and compliance tracking.

⚙️

Role-Based Access

Employee, Manager, and Admin roles with fine-grained permission control.

Getting Started

Option 1: Using Docker (Recommended)

# Clone repository git clone https://github.com/khushboosingh-coder/TPS.LM.git cd TPS.LM # Start all services docker-compose up # Visit the application http://localhost:3000

Option 2: Local Development

# Terminal 1: Database Setup psql -U postgres -f database/init.sql # Terminal 2: Backend Server cd backend npm install npm run dev # Terminal 3: Frontend App cd frontend npm install npm run dev # Visit the application http://localhost:3000

URL Endpoints

Service URL Description
